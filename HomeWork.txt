
   Linux terminal (GitBash) commands

1.Посмотреть где я.
       command: pwd
2.Создать папку.
       command: mkdir 26g2
3.Зайти в папку.
       command: cd 26g2
4.Создать 3 папки. 
       command: mkdir home_1 home_2 home_3
5.Зайти в любую папку.
       command: cd home_1
6.Создать 5 файлов (3 txt, 2 Json).
       command: touch test_1.txt test_2.txt test_3.txt f_1.json f_2.json
Или 
       command: touch new_file{1..3}.txt new_file{1..2}.json
7.Создать 3 папки.
       command: mkdir work_1 work_2 work_3
 или 
       command: mkdir {1..3}
8.Вывести список содержимого папки.
       command: ls -la
9.Открыть любой txt файл.
       command: vim test_1.txt
10.Написать туда что-нибудь, любой текст.
Hello
My name is Luda
I live in Kharkiv
I have a cat
It name is Elis
11.сохранить и выйти. 
       command: esc; 
       command: wq
12.Выйти из папки на уровень выше.
       command: cd ..
13. Переместить любые 2 файла, которые вы создали, в любую другую папку. 
       command: mv ./home_1/{test_1.txt,f_1.json} ./home_3  
14.Скопировать любые 2 файла, которые вы создали, в любую другую папку.
       command: cp ./home_3/{test_1.txt,f_1.json} ./home_2
15.Найти файл по имени.
       command: find . -name "test_1.txt"
16.Просмотреть содержимое в реальном времени (команда grep) изучите как она работает.
       command: tail -F home_1/test_1.txt
17.Вывести несколько первых строк из текстового файла.
       command: head -2 home_1/test_1.txt
18.Вывести несколько последних строк из текстового файла.
       command: tail -2 home_1/test_1.txt
19.Просмотреть содержимое длинного файла (команда less) изучите как она работает.
       command: less home_1/test_1.txt
20.Вывести дату и время.
       command: date
